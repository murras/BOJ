## [BOJ 10987번 단위 문제](https://noj.am/10987)

간단한 문제다.
string을 입력받고 인덱스로 한글자씩 살피면서 `'a' 'e' 'i' 'o' 'u'`이면 count값을 1씩 증가시킨다. 모두 살핀 다음 출력하면 된다.

```cpp
#include <iostream>
using namespace std;

bool check(char c) {
    if (c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u') {
        return true;
    }
    return false;
}

int main() {
    string s;
    cin >> s;
    int stringSize = s.size();
    int count = 0;
    for (int i = 0; i < stringSize; i++) {
        if (check(s[i])) {
            count++;
        }
    }
    cout << count;
}
```
