## [BOJ 10539번 수빈이와 수열 문제](https://www.acmicpc.net/problem/10539)

수열 B가 3, 2, 3, 5일 떄

1) 3 = 3/1

2) 2 = (3+x)/2 에서 x = 1

3) 3 = (3+x+y)/3 에서 y = 5

4) 5 = (3+x+y+z)/4 에서 z = 11


> 10539.cpp

```cpp
#include <bits/stdc++.h>
using namespace std;
int arr[101];
int res[101];
int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> arr[i];
    }
    int plus = 0;
    res[1] = arr[1];
    for (int i = 2; i <= n; i++) {
        plus = 0;
        for (int j = 1; j <= i; j++) {
            plus += res[j];
        }
        res[i] = arr[i] * i - plus;
    }
    for (int i = 1; i <= n; i++) {
        cout << res[i] << ' ';
    }
}

```
